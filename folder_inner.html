논문 제목

"퍼소나 스토리" 종합요약

“개인정보 유형의 정보 민감도에 따른 모바일 서비스 UI 디자인 프릭션 분석” 논문은 이용자가 입력·제공하는 개인정보의 민감도 수준에 따라 UI 상에서 적절한 프릭션(friction)을 설계하는 것을 목표로, 입력 절차·경고 문구·시각적 강조 요소 등을 중심으로 디자인 가이드라인을 제시하였다. 반면 “고령 사용자를 위한 모바일 결제 UI 접근성 향상 방안 연구” 논문은 고령층 사용자가 모바일 결제 서비스를 이용할 때 발생하는 인지적·물리적 부담을 최소화하는 것을 목표로, 버튼 크기·글자 가독성·단계 단순화 등을 중심으로 디자인 가이드라인을 설계했다는 차별점이 존재한다. 반면 두 논문 모두 사용자의 특성과 맥락을 고려하여 UI 프릭션 또는 접근성 요소를 조정해야 한다는 필요성과, 디자인 가이드라인을 실제 서비스 시나리오에 적용·검증하는 과정의 중요성을 공통적으로 확인하였다.
